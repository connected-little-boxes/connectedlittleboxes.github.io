Server

Backend Support

Server support for a Connected Little Box makes things even more interesting. You don't need it. A box will quite happily talk to another box with no server needed. But if you want to manage and deploy your boxes you might find it useful to have a single point of control. That's what the server is all about.

The server software is still under development. That's software-speak for 'It seems to work for us but we want to make sure it will work for you.' We are running our own little server at Connected Humber. If you want to join in with us and help us build the system, head to the contacts and get in touch with us.
